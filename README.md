# MH-Design

> A minimal CSS design system built from scratch.

MH-Design is a lightweight modular CSS framework focused on clean architecture,
reusable utilities, and component-based structure.

---

## Installation

### Clone the repository:

```bash
git clone https://github.com/mennahsalah60-coder/MH-design.git
```

then add in your project:
    <link rel="stylesheet" href="./styles/mh-design.css">

Or include the CSS files manually in your project:
<link rel="stylesheet" href="styles/variables.css">
<link rel="stylesheet" href="styles/base.css">
<link rel="stylesheet" href="styles/layout.css">
<link rel="stylesheet" href="styles/utilities.css">
<link rel="stylesheet" href="styles/components/buttons.css">
<link rel="stylesheet" href="styles/components/cards.css">
<link rel="stylesheet" href="styles/components/navbar.css">
<link rel="stylesheet" href="styles/components/forms.css">

### Core Concepts

1. Design Tokens
All styling is powered by CSS variables:
Colors
Spacing system
Typography scale
Border radius
Transitions

2. Layout System
Container
<div class="mh-container">
  Content
</div>
Responsive max-width system included.

4. Utilities
Utility-first helpers included:
Spacing classes
Flex helpers
Text alignment
Width / Height helpers

3. Components
a. Button
b. Cards
c. Form System
S. Navbar
